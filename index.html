Bonjour, je suis Erwann RIVAUD

Apprenti Intégrateur Web chez OpenClassrooms depuis 1 an et demi, je conçois et développe des sites web en respectant les bonnes pratiques du développement et du référencement.

De l’analyse du cahier des charges à la mise en ligne, je prends en charge l’intégration des interfaces utilisateur en garantissant une expérience fluide et accessible sur tous les supports. Je maîtrise HTML et CSS pour structurer et styliser les pages web, ainsi que JavaScript et React pour créer des interfaces dynamiques et interactives. Grâce à mon expertise en responsive design, j’assure une adaptation optimale des sites sur mobiles, tablettes et desktops. Passionné par le web et les nouvelles technologies, je suis toujours à la recherche de nouvelles compétences pour perfectionner mon code et optimiser l’expérience utilisateur.

Mes compétences

HTML et CSS

En tant qu’intégrateur web, je maîtrise HTML et CSS , deux piliers fondamentaux du développement web moderne. Ces technologies me permettent de concevoir des interfaces web structurées, esthétiques et adaptées aux standards actuels.

JavaScript

JavaScript est un langage indispensable dans le développement web moderne, permettant d’ajouter interactivité, dynamisme et réactivité aux interfaces utilisateur. En tant qu’intégrateur web, je l’utilise pour améliorer l’expérience utilisateur et optimiser les performances des sites web.

React

React est une bibliothèque JavaScript puissante et flexible qui révolutionne la façon de concevoir les interfaces web. Grâce à ma maîtrise de React.js, je développe des applications rapides, interactives et modulaires, offrant une expérience utilisateur fluide et performante.
